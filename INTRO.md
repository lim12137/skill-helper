# 项目简介

Skill Helper 是一个“技能平台化”项目，用来解决以下问题：

- 让用户在网页中直接使用技能
- 让用户协作修改或创建技能
- 让技能具备版本管理、权限控制与可发布能力

当前第一版实现为 `copaw-skill-platform`，重点验证：

1. 多用户 + 登录鉴权
2. 技能 CRUD + 版本历史
3. 运行任务队列（异步）
4. Docker 一键部署
5. GitHub Action 多架构镜像构建（amd64/arm64）

适用场景：

- 团队内部 AI 助手技能中心
- 个人/组织的技能市场原型
- 需要“可管理、可迭代、可审计”的技能工程体系
